Manage speaking sessions between students and teachers integrated with Discord.

== Description ==

This plugin allows students to purchase speaking session packages and book sessions with available teachers. Sessions are conducted on Discord with automated room creation and credit deduction.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dnd-speaking` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure settings in DND Speaking > Settings.
4. Create WooCommerce products with speaking sessions meta field.
5. Use shortcodes: [dnd_speaking_dashboard], [dnd_teacher_list]

== Usage ==

- Students: Purchase sessions, view credits, book teachers.
- Teachers: Toggle availability, approve bookings.
- Admins: Configure Discord bot, session duration.
