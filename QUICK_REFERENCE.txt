/**
 * DND Speaking WooCommerce - Quick Reference
 * 
 * CREATING A LESSON PACKAGE PRODUCT:
 * ==================================
 * 
 * 1. Go to: WooCommerce → Products → Add New
 * 
 * 2. Product Data Metabox:
 *    - Select "DND Speaking" from the dropdown
 * 
 * 3. General Tab (will appear automatically):
 *    - Regular Price: $XX.XX (required)
 *    - Sale Price: $XX.XX (optional)
 *    - Schedule: From/To dates for sale (optional)
 *    - Tax Status: Taxable/Shipping only/None
 *    - Tax Class: Standard or custom tax class
 *    - Amount (Lesson Sessions): Number of lessons (e.g., 5, 10, 20)
 * 
 * 4. Fill in standard product details:
 *    - Product title
 *    - Product description
 *    - Product image
 * 
 * 5. Publish the product
 * 
 * 
 * WHAT HAPPENS WHEN A CUSTOMER BUYS:
 * ==================================
 * 
 * 1. Customer adds product to cart (sees lesson count)
 * 2. Customer completes checkout
 * 3. Order status changes to "Completed"
 * 4. System automatically:
 *    - Calculates total sessions: Amount × Quantity
 *    - Adds sessions to customer's account
 *    - Logs the transaction
 *    - Adds note to order
 * 
 * 
 * EXAMPLE PRODUCTS:
 * ================
 * 
 * Starter Package:
 *   Regular Price: $50
 *   Amount: 5 lessons
 * 
 * Standard Package:
 *   Regular Price: $90
 *   Sale Price: $80
 *   Amount: 10 lessons
 * 
 * Premium Package:
 *   Regular Price: $200
 *   Sale Price: $170
 *   Amount: 25 lessons
 * 
 * 
 * CHECKING CREDITS ADDED:
 * ======================
 * 
 * 1. View the order in WooCommerce → Orders
 * 2. Look for order note: "Added X lesson session(s) to user account"
 * 3. Check error log at: /wp-content/debug.log
 * 4. Query database table: wp_dnd_speaking_credits
 * 
 * 
 * TROUBLESHOOTING:
 * ===============
 * 
 * Credits not added?
 *   - Verify order status is "Completed"
 *   - Check product type is "DND Speaking"
 *   - Review order notes
 *   - Enable WordPress debug logging
 * 
 * Fields not showing?
 *   - Ensure WooCommerce is active
 *   - Select "DND Speaking" product type
 *   - Clear cache and refresh page
 * 
 * 
 * KEY FEATURES:
 * ============
 * 
 * ✓ Custom "DND Speaking" product type
 * ✓ All standard WooCommerce pricing features
 * ✓ Tax configuration support
 * ✓ Sale scheduling
 * ✓ Custom "Amount" field for lesson sessions
 * ✓ Automatic credit addition on order completion
 * ✓ Quantity support (buy multiple packages)
 * ✓ Prevents duplicate credit additions
 * ✓ Full logging for tracking
 * ✓ Display on product page, cart, and checkout
 * 
 * 
 * SUPPORTED ORDER STATUSES:
 * ========================
 * 
 * Credits are added when order status becomes "Completed"
 * 
 * If you need credits added on other statuses (like "Processing"),
 * you can modify the hook in class-woocommerce.php:
 * 
 * Change:
 *   add_action('woocommerce_order_status_completed', ...)
 * 
 * To:
 *   add_action('woocommerce_order_status_processing', ...)
 *   add_action('woocommerce_order_status_completed', ...)
 */
